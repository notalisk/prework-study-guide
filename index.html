HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is used to write the actual content of a webpage.
  • In order to make a website more accessible, add alt text to every image.
  • The 'iframe' element can be used to add videos to a website.
  • Websites can have built in forms, using the 'form' element.

CSS

  • Margin is the space around the outside of an element
  • Padding is the space around the content inside of an element
  • Select a class by typing a "." followed by the class name
  • Select an id by typing "#" followed by the id
  • Use the 'box-shadow' property to make a shadow around an element.
  • To make something scale depending on the size of the browser window, use the 'vw' (viewport width) unit.

Git

  • all Git commands start with the word 'git'
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls the most recent version of main branch from GitHub
  • git add -A: adds all modifications to the staging area, to prepare for the next commit
